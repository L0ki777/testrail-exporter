TestRail Step Exporter

Export test case steps as clean, readable text — no HTML tags

How to get your API Key

▼
1
Log in to your TestRail instance (e.g. https://yourcompany.testrail.io)
2
Click on your name in the top right corner (e.g. "Lokesh Nat...")
3
Select "My Settings" from the dropdown menu
4
Click the "API KEYS" tab at the top of the settings page
5
Click "Add Key" and enter a name like Export Steps
6
Click "Save Configuration"
7
Copy the generated API key and paste it in the field below. Store it somewhere safe — you won't be able to see it again in TestRail.
Security tip: Your API key is never stored by this tool. It's only used for the current session to fetch data. You can delete the key from TestRail after you finish exporting.
