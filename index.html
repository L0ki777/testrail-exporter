How to get your API Key

▼
1
Log in to your TestRail instance (e.g. https://yourcompany.testrail.io)
2
Click on your name in the top right corner
3
Select "My Settings" from the dropdown menu
4
Click the "API KEYS" tab at the top of the settings page
5
Click "Add Key" and enter a name like Export Steps
6
Click "Save Configuration"
7
Copy the generated API key and paste it in the field below.
8
(For Import) Run a local CORS proxy — install once with npm i -g local-cors-proxy, then run lcp --proxyUrl https://yourcompany.testrail.io and paste http://localhost:8010/proxy into the CORS Proxy field above.
Security tip: Your API key is never stored by this tool. It's only used for the current session to fetch data.
